# হ্যালো ওয়ার্ল্ড!

প্রায় সব প্রোগ্রামরের প্রোগ্রামিং এ হাতেখড়ি হয় হ্যালো ওয়ার্ল্ড প্রোগ্রাম দিয়ে।
হ্যালো ওয়ার্ল্ড এমন একটি সহজ প্রোগ্রাম যেখানে কম্পিউটার কনসলে `Hello World!` লেখাটি প্রিন্ট করা হয়।

চলুন এখন সুইফট দিয়ে একটি হ্যালো ওয়ার্ল্ড প্রোগ্রাম লেখা যাক।

## সুইফট "Hello, World!" প্রোগ্রাম

```Swift
// সুইফট "Hello, World!" প্রোগ্রাম

print("Hello, World!")

```

### আউটপুট

```
Hello, World!
```

:::tip

**নোট:** প্রোগ্রামটি আপনার কম্পিউটারে রান করেতে হলে অবশ্যই আপনার কম্পিউটারে সুইফট ইন্সটল করা থাকতে হবে, যদি না থাকে তাহলে [ইন্সটল](/docs/intro#getting-started) অংশ দেখতে পারেন।

:::

## সুইফট "Hello, World!" কিভাবে কাজ করে?

উপরে উল্লেখিত প্রোগ্রামে দুইটি অংশ আছে।

### ১. সুইফট কমেন্ট

```
// সুইফট "Hello, World!" প্রোগ্রাম
```

দুইটি স্লাশ `//` দিয়ে সুইফট প্রোগ্রামের কমেন্ট শুরু হয়ে থাকে।

কমেন্ট মূলত প্রোগ্রাম দিয়ে কি কাজ করা হয়েছে বা কোন লাইন দিয়ে কি করা হয়েছে সেটির বর্ণনা। পরবর্তী সময় যখন আবার এই কোড পড়ার দরকার পরে তখন সহজেই বোঝা যাই, অথবা অন্য প্রোগ্রামাররা যখন আপনার লেখা কোড বুঝতে চেষ্টা করবে তখন এটা অনেক কাজের।

কমেন্ট অনেক রকম হয়ে থাকে, বিস্তারিত যানতে [কমেন্ট](/docs/intor/comments) অংশ দেখতে পারেন।

### ২. প্রিন্ট স্টেটমেন্ট

```
print("Hello, World!")
```

উপরের কোডটি দিয়েই মূলত `Hello, World!` লেখাটি কম্পিউটার স্ক্রীনে প্রিন্ট করে হয়েছে। কিভাবে হয়েছে প্রবর্তিতে এটি নিয়ে বিস্তারিত আলোচনা করে হয়েছে।

অভিনন্দন! আপনি সফলভাবে আপনার প্রথম সুইফট প্রোগ্রাম রান করতে পেরেছেন।

পরবর্তী সেকশনে সুইফট ভ্যারিয়েবল, কনস্টান্টস এবং লিটারাল নিয়ে আলোচনা করা হবে।
